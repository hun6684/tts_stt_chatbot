# 🍻 알코봇의 시련 : AI 음성 술게임 마스터

> "술게임 마스터가 되고 싶은가, 자네...? 나부터 이겨보시지!"
> 음성 인식(STT)과 음성 합성(TTS)을 활용해 무자비한 AI와 1:1로 대결하는 보이스 술게임 오락실입니다.

## 🎮 게임 라인업
1. **🍦 베스킨라빈스 31**: AI와 1~3개의 숫자를 번갈아 부르며 31을 피하는 눈치 게임
2. **🚇 2호선 호선 게임**: 51개 역이 담긴 풀버전 DB를 바탕으로 진행되는 2호선 역 이름 대기
3. **📜 사자성어 이어 말하기**: AI가 부르는 앞 두 글자를 듣고 3초 안에 나머지 두 글자 외치기
4. **🍓 딸기 게임**: 1~8개까지 박자에 맞춰 "짝"과 "딸기"를 정확한 발음과 띄어쓰기로 외치는 리듬 게임
