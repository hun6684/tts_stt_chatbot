# 🍻 알코봇과의 대결 : AI 음성 술게임

> "술게임 마스터가 되고 싶은가, 자네...? 나부터 이겨보시지!"
> 음성 인식(STT)과 음성 합성(TTS)을 활용해 무자비한 AI와 1:1로 대결하는 보이스 술게임 오락실입니다.
<br>
## 📢 프로젝트 소개 (About)
텍스트를 타자로 치는 뻔한 웹 게임은 그만! **오직 '목소리'와 '박자감'만으로 깐깐한 AI 알코봇과 대결하는 신개념 인터랙티브 음성 게임**입니다. 
혼술할 때 심심풀이로, 혹은 진짜 술자리에서 지지 않기 위한 맹훈련용으로 기획되었습니다. 사용자의 발음과 박자를 실시간으로 분석하며, 상황에 따라 알코봇의 얄미운(?) 티키타카를 경험할 수 있습니다.
<br>
## 🎮 게임 라인업
1. **🍦 베스킨라빈스 31**: AI와 1~3개의 숫자를 번갈아 부르며 31을 피하는 눈치 게임
2. **🚇 지하철 게임**: 뿌뿌 ~ 칙칙 폭폭 51개의 2호선 역 이름 대기
3. **📜 사자성어 이어 말하기**: AI가 부르는 앞 두 글자를 듣고 3초 안에 나머지 두 글자 외치기
4. **🍓 딸기 게임**: 1~8개까지 박자에 맞춰 "짝"과 "딸기"를 정확한 발음과 띄어쓰기로 외치는 리듬 게임

## ✨ 핵심 기능 및 디테일
- **Hands-Free Play**: 키보드나 마우스 조작 없이 마이크 하나로 처음부터 끝까지 게임 진행
- **STT 깐깐 판정 로직**: '딸기 게임' 등의 오디오 게임 시, 띄어쓰기 공백을 제거하고 순수 텍스트 배열만 비교하여 음성 인식의 빈틈을 악용하는 꼼수 원천 차단
- **다이나믹 페르소나**: AI가 이기거나 유저가 틀렸을 때, 평소와 다른 낮고 묵직한 목소리 톤(Onyx 보이스)으로 "마시세요!"를 외치는 디테일 적용
